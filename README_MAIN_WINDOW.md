# 主窗口新功能说明

## 🎯 新增功能概述

主窗口现在包含以下新功能：

### 1. 📱 设备管理标签页
- **设备列表管理**：显示所有配置的设备信息
- **设备信息**：包含设备名称、屏幕宽度、屏幕高度、User Agent
- **添加设备**：可以添加新的设备配置
- **删除设备**：删除不需要的设备配置
- **默认设备**：预设了4种常见设备类型

### 2. 🌐 URL管理标签页
- **URL输入**：可以添加新的URL地址
- **URL列表**：显示所有配置的URL
- **添加URL**：添加新的URL到列表
- **删除URL**：删除不需要的URL
- **随机选择**：随机选择一个URL
- **默认URL**：预设了5个常用网站

### 3. 🚀 运行功能增强
- **随机设备选择**：点击运行时会随机选择一个设备
- **随机URL选择**：同时随机选择一个URL
- **WebView弹窗**：创建新的WebView弹窗显示网页
- **设备信息应用**：自动应用选中设备的屏幕尺寸和User Agent

## 🔧 功能详解

### 设备管理功能

#### 默认设备列表
1. **Desktop Windows**
   - 分辨率：1920x1080
   - User Agent：Chrome Windows

2. **Desktop macOS**
   - 分辨率：1440x900
   - User Agent：Chrome macOS

3. **Mobile Android**
   - 分辨率：360x640
   - User Agent：Chrome Android

4. **Mobile iOS**
   - 分辨率：375x667
   - User Agent：Safari iOS

#### 添加自定义设备
1. 在"设备名称"输入框中输入设备名称
2. 设置屏幕宽度和高度
3. 输入User Agent字符串
4. 点击"添加设备"按钮

#### 删除设备
1. 在设备列表中找到要删除的设备
2. 点击对应行的"删除"按钮
3. 确认删除操作

### URL管理功能

#### 默认URL列表
- https://www.baidu.com
- https://www.google.com
- https://www.bing.com
- https://www.github.com
- https://www.stackoverflow.com

#### 添加新URL
1. 在URL输入框中输入网址
2. 点击"添加URL"按钮
3. URL会自动添加到列表中

#### 随机选择URL
1. 点击"随机选择URL"按钮
2. 系统会随机选择一个URL并显示

### 运行功能

#### WebView弹窗创建
1. 点击"运行"按钮
2. 系统自动：
   - 随机选择一个设备
   - 随机选择一个URL
   - 创建WebView弹窗
   - 应用设备的屏幕尺寸和User Agent

#### WebView弹窗特性
- **设备信息显示**：顶部显示设备名称、分辨率、URL
- **自适应尺寸**：根据设备分辨率设置弹窗大小
- **User Agent应用**：自动应用设备的User Agent
- **非模态设计**：可以同时打开多个弹窗
- **控制按钮**：刷新和关闭按钮

## 🚀 使用方法

### 基本操作流程
1. **配置设备**：
   - 切换到"设备管理"标签页
   - 添加或修改设备配置
   - 设置屏幕尺寸和User Agent

2. **配置URL**：
   - 切换到"URL列表"标签页
   - 添加需要访问的URL
   - 管理URL列表

3. **运行测试**：
   - 点击"运行"按钮
   - 查看自动创建的WebView弹窗
   - 验证设备配置是否正确

### 高级功能
- **批量测试**：可以多次点击运行，创建多个不同设备的弹窗
- **实时配置**：修改设备或URL后立即生效
- **灵活管理**：支持添加、删除、修改所有配置项

## 📋 系统要求
- Python 3.6+
- PyQt5 >= 5.15.0
- PyQtWebEngine >= 5.15.0

## 🔍 测试方法
运行测试脚本：
```bash
python3 test_main_window.py
```

## 💡 技术特点
- **随机化算法**：智能随机选择设备和URL
- **动态配置**：支持运行时添加和删除配置
- **WebView集成**：完整的网页浏览功能
- **设备模拟**：真实模拟不同设备的显示效果
- **User Agent管理**：完整的浏览器标识管理

## 🐛 已知问题
- 某些网站可能因为安全策略无法在WebView中加载
- 复杂的JavaScript应用可能需要等待加载完成
- 高分辨率设备在小屏幕上可能显示不完整
